# ゲームはここでプレイ↓
(https://waai94.github.io/74GameJam/)

## 概要
ワイヤーを地形にくっつけてどんどん上を目指しましょう!
画面外に行く、または赤いラインに触れるとゲームオーバです。

## 操作説明
左クリック:ワイヤーを飛ばす
右クリック:アイテムを使う

## 詳細ルール
### プレイヤーの動作
フックはマウスを長押ししてる間使えます。終点に行くと慣性が消えます。途中で離すと慣性が残るのでうまく使い分けるのがコツです。
フックは着地のたびに2回使えます。
### アイテム
スーパージャンプ: 真上に飛びます。フック回数は復活しません。
フック回復: フック使用可能数を1増やします
